🚀 Pine Script AI Generator

A Streamlit-based web application that generates TradingView Pine Script v6 indicators from natural language prompts.

📌 Project Overview

This project allows users to generate complete TradingView Pine Script indicators simply by describing their strategy in plain English.

The system converts user prompts into fully working Pine Script v6 code that can be directly used in TradingView charts.

✨ Features
Generate Pine Script v6 indicators using AI prompts
Simple Streamlit web interface
Instant code generation
Copy & paste ready output
Alert conditions and signal logic support
Fast and lightweight deployment
🛠️ Technologies Used
Python
Streamlit
Google Gemini API
Pine Script v6
OpenPyXL (for Excel-based workflows in extended version)
⚙️ How It Works
User enters a natural language prompt (indicator description)
App processes the request
AI generates Pine Script v6 code
Output is displayed in the UI
User copies code into TradingView
💡 Example Prompt

Create a Pine Script v6 indicator that:

Uses 50 EMA trend filter
Generates buy signal when RSI crosses above 30
Generates sell signal when RSI crosses below 70
Shows arrows on chart
Includes alert conditions
